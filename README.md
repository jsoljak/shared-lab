# shared-lab

A place where I keep things worth sharing — notes, tools, ideas,
and occasional experiments. Mostly for friends and people I talk to.

This is not a portfolio or a blog. It's a shared shelf —
I put things here so I can send a link instead of an attachment.

No expertise claims. No agenda. Just stuff I find useful or interesting.

## Structure

tools/     — prompts, templates, scripts
